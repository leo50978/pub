# Contexte du projet

## Vision generale

Le projet est une experience visuelle interactive pensee pour etre enregistree en video verticale, principalement sur mobile, afin de promouvoir des services de creation de sites web modernes.

Il ne s'agit pas d'un site web classique a parcourir, mais d'une scene web cinematographique construite pour etre capturee en screen recording et transformee en video publicitaire courte, autour de 30 secondes.

## Intention creative

L'objectif est de montrer :

- une forte maitrise du design web moderne
- une capacite a produire des experiences immersives
- une direction artistique minimaliste mais marquante
- une maitrise des animations avancees
- la capacite a imaginer plusieurs types de sites pour plusieurs secteurs

Le projet repose sur trois piliers :

- minimalisme visuel
- animations cinematographiques
- narration visuelle forte avec peu de texte

## Structure de l'experience

L'experience est composee de trois scenes principales.

### Scene 1 - Introduction

L'ouverture se fait dans un univers sombre, immersif, presque spatial.

Des mots apparaissent pour evoquer le processus creatif :

- Creer
- Elever
- Captiver
- Mesurer
- Marquer
- Transformer

Le premier mot apparait lettre par lettre.
Les suivants apparaissent avec une impression de profondeur.
Ensuite, les mots reculent dans l'espace comme si la camera avançait dans un univers d'idees.

Le dernier mot, `Transformer`, reste au centre puis declenche une transition organique en forme d'onde, proche d'une goutte d'eau, qui transforme le fond sombre en interface claire.

### Scene 2 - Traversee des interfaces

Une premiere interface complete apparait, puis s'eloigne en legere perspective 3D pour reveler un ensemble d'autres interfaces disposees dans l'espace.

La camera virtuelle traverse ensuite une galerie d'interfaces web representant plusieurs types de projets :

- landing page moderne
- site e-commerce
- site universitaire
- portfolio creatif
- interface SaaS / application
- site corporate / business

Chaque interface est placee avec de la profondeur sur l'axe Z et un angle legerement diagonal pour renforcer l'impression d'immersion.

Quand la camera arrive devant une interface :

- l'interface se redresse un peu face a la camera
- un mini scroll automatique montre une partie du contenu
- la camera poursuit sa trajectoire vers la suivante

Le fond reste vivant avec des mouvements subtils de zoom et de translation.

### Scene 3 - Conclusion et appel a l'action

Un curseur drag-and-drop apparait depuis le bas de l'ecran et tire une surface blanche qui recouvre progressivement l'ecran.

Cette transition amene un ecran final avec :

- une phrase animee lettre par lettre : `Pour les entreprises qui veulent se demarquer`
- un bouton d'appel a l'action : `Contactez-nous`

Le curseur devient ensuite un curseur de clic, se deplace vers le bouton et simule un clic avec un effet de hover.

La fin doit etre claire, elegante et orientee conversion.

## Contraintes techniques

Le projet doit etre developpe avec :

- HTML
- CSS / Tailwind
- JavaScript
- GSAP
- Google Fonts

L'objectif est d'obtenir une experience fluide, legere et completement controlee sans moteur 3D complexe.

## Format cible

Le rendu est pense pour :

- un format vertical mobile
- un affichage optimise iPhone XR
- un usage en screen recording
- une video publicitaire courte et immersive

## Philosophie

Le projet ne cherche pas seulement a montrer un site web.
Il cherche a montrer que le web peut etre :

- une experience visuelle
- une forme d'art digital
- un outil de communication puissant

L'idee centrale est qu'un site web peut captiver, marquer et transformer l'image d'une marque.

## Ce que je retiens pour la suite

Quand on commencera la production, il faudra preserver ces priorites :

- privilegier l'impact visuel plutot qu'une navigation classique
- penser l'ensemble comme une sequence video choregraphiee
- garder peu de texte a l'ecran
- soigner les transitions entre scenes
- maintenir une qualite mobile premium, fluide et cinematographique
